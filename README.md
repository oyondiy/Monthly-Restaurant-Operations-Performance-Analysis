## 📖 Project Overview

This project analyzes operational and financial performance across a multi-brand restaurant chain. The solution combines relational database design, SQL-based analytics, and Power BI visualization to provide actionable insights into restaurant performance, profitability, customer behavior, and sales trends.

The dashboard enables stakeholders to:

* Monitor revenue and profitability across restaurant locations
* Track customer purchasing patterns and payment preferences
* Identify top-performing and underperforming menu items
* Compare restaurant performance across cities
* Analyze sales trends over time
* Support data-driven operational decision-making

## 🔍 SQL Analytics

The project includes a collection of analytical SQL queries developed to answer real-world business questions.

### Revenue Analysis

Calculate total revenue generated by each restaurant.

```sql
SELECT
    restaurant_name,
    SUM(total_amount) AS total_revenue
FROM orders
GROUP BY restaurant_name
ORDER BY total_revenue DESC;
```

### Profitability Analysis

Calculate profit generated by each menu item.

```sql
SELECT
    f.food_item,
    SUM(o.total_amount) AS revenue,
    SUM(c.total_cost * o.quantity) AS total_cost,
    SUM(o.total_amount - (c.total_cost * o.quantity)) AS profit
FROM orders o
JOIN food_items f
    ON o.food_item = f.food_item
JOIN cost c
    ON f.food_item = c.food_item
GROUP BY f.food_item;
```

## 📐 DAX Measures

### Total Revenue

```dax
Total Revenue =
SUM(orders[total_amount])
```

### Total Cost

```dax
Total Cost =
SUMX(
    orders,
    orders[quantity] * RELATED(cost[total_cost])
)
```

### Profit

```dax
Profit =
[Total Revenue] - [Total Cost]
```

### Profit Margin %

```dax
Profit Margin % =
DIVIDE([Profit], [Total Revenue], 0) * 100
```

### Average Order Value

```dax
Avg Order Value =
AVERAGEX(
    orders,
    orders[total_amount]
)
```

### Total Customers

```dax
Total Customers =
DISTINCTCOUNT(orders[customer_name])
```
